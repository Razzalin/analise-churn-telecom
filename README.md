# 📊 Análise de Churn - Telecom

## 📌 Objetivo

Este projeto tem como objetivo analisar os fatores que influenciam o cancelamento (churn) de clientes em uma empresa de telecomunicações, utilizando análise exploratória de dados (EDA).

---

## 📂 Base de Dados

A base contém informações demográficas, contratuais e de serviços contratados pelos clientes.

Principais variáveis analisadas:

- Tipo de contrato
- Tempo de permanência (tenure)
- Valor da mensalidade
- Método de pagamento
- Perfil demográfico

---

## 🔎 Etapas do Projeto

1. Importação e exploração inicial dos dados
2. Tratamento de colunas estruturadas (JSON)
3. Transformação da variável churn
4. Análise exploratória
5. Visualização de dados
6. Análise de correlação
7. Extração de insights estratégicos

---

## 📊 Principais Insights

- Clientes com contrato mensal apresentam maior taxa de churn.
- Contratos de longo prazo demonstram alta retenção.
- Clientes nos primeiros meses são mais propensos a cancelar.
- Mensalidades mais elevadas apresentam associação com maior churn.

---

## 🧠 Técnicas Utilizadas

- Python
- Pandas
- Numpy
- Seaborn
- Matplotlib
- Análise de Correlação (Pearson)

---

## 🎯 Conclusão

A análise identificou padrões claros de evasão, permitindo direcionar estratégias de retenção, como incentivo a contratos de longo prazo e monitoramento de clientes em fase inicial.
